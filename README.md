## RoastHub Frontend
- Used React Next.js to build the front end for [RoastHub](https://github.com/guanjie-tom-wang/RoastHub/)
## How to run
- Clone this repo and cd it into the folder
- Install the requirement: `npm install`
- Run this project by `npm run dev`
- Access this frontend by typing `http://localhost:8080`
